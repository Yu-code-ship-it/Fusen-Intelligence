# Fusen Intelligence: An AI-powered Meaning Generation and Decision-Making Support System
## Summary
Fusen Intelligence is an AI-powered system that supports meaning generation, self-understanding, and decision-making by recording small insights, feelings, conditions, and values as “fusen,” or sticky-note-like units of meaning.
The system helps users visualize relationships between these notes and generate new meanings or statements by referring to multiple notes together.
## Background
People have many insights, ideas, emotions, and values. However, many of them slip away and are lost.
To make these insights reusable, and to use them as a first step toward understanding people who find communication difficult, I want to build a system that makes it possible to visualize the connections between meanings and thought processes.
This theme is important to me because I believe that small, unclear feelings or ideas can become meaningful when they are recorded, connected, and reflected upon.
## Data and AI Technologies
The data used in this project includes short notes created by users, tags, categories, chronological information, and past selection histories. In this project, I call these small units of meaning “fusen.”
The project begins with each user creating and recording fusen.
The basic process is:
1. Referencing fusen and forming relationships between them2. Generating new fusen or comments3. Evaluating and sorting meanings4. Saving the results as a structure
This process is similar to an “autopoietic structure” or autopoiesis, in which a system reproduces itself using its own elements.
Useful AI technologies may include natural language processing, text classification, similarity search, clustering, summarization, generative AI, and recommendation systems.
## How It Will Be Used
The created fusen can be classified as follows:
- States, such as “rain” or “tired”- Desires, such as “want to rest” or “want to eat”- Constraints, such as “work” or “diet”- Values, such as “want to prioritize health”
These fusen do not exist in isolation. By referring to one another, they can generate new meanings, statements, or decisions.
For example, suppose the following fusen exist:
- “Rain”- “Want to relax at home”- “There are donuts”- “On a diet”- “Have work”
By referring to these fusen together, the system might generate a statement such as:
> “I’ll stay home, do a little work, go easy on the donuts, and exercise.”
This demonstrates that decision-making does not come from a single factor, but is generated through the combination of multiple conditions.
I hope to use this system first to help reduce the sense of isolation felt by children who find it difficult to communicate with the outside world. It may help make visible the frustration, loneliness, or confusion that children themselves cannot fully explain.
It could also support teachers and caregivers by helping them avoid dismissing such feelings as mere selfishness or misunderstanding.
## Challenges
In this system, several elements are crucial:
- Weights that determine which fusen should be prioritized- Relationships between fusen, such as complementarity or conflict- History that reflects past selection trends- Selection criteria that determine how generated content should be retained
The cold start problem is also likely to be a major challenge. To address this, the system may need a way to identify an individual’s values through a wide range of questions.

One approach is to avoid asking users to describe their values directly. Instead, the system could present concrete scenarios and ask users to choose how they would respond. By analyzing patterns in word choice, recurring expressions, and responses to hypothetical situations, the system can infer values inductively — without requiring users to articulate them explicitly.
Another important issue is how to store and protect personal information.

 Since this system may handle sensitive emotional or personal data, privacy and data security must be carefully considered.
This system should not replace the judgment of teachers, caregivers, or support staff. It should be used only as an auxiliary tool to help understand a child’s condition and background.
## Next Steps
This concept combines:
- A mechanism for generating thoughts- A mechanism for selecting thoughts
In the future, it may develop into:
- A meaning-generation system- A decision-making support model
As a next step, I would like to create a simple prototype in which users can create fusen, classify them, and see possible relationships between them.
